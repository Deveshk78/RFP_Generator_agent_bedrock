# Contributing

Thanks for considering contributions! Please follow these steps:

- Fork the repository and create a feature branch
- Run tests and linters locally (if applicable)
- Open a pull request with a clear description of changes

Coding style
- Python: follow PEP8
- Frontend: follow existing formatting rules (Prettier, Tailwind)

Reporting issues
- Use the repository Issues tab to report bugs or request features. Provide steps to reproduce and relevant logs.
